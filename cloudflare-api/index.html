HTTP Cat mirror

This is a personal fallback mirror of the image API from http.cat, used only in case the upstream service is unavailable. It is not affiliated with or endorsed by the original project.

All credit for the concept, artwork, and status descriptions goes to the original httpcats/http.cat project and its contributors.

Usage: request any HTTP status code as a path, e.g. /200 or /404.jpg, to get the matching cat image.
